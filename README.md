# Webscraper-timeMachine
Web scraper that pulls data from Coinbase's archived "about" pages 

## Prerequisites

You will need the following things properly installed on your computer.

* [Git](http://git-scm.com/)
* [Node.js](http://nodejs.org/) (with NPM)

## Installation

* `git clone <repository-url>` this repository
* `cd webscraper-timeMachine`
* `npm install`

## Running 

* `node server.js`
* Visit your app at [http://localhost:8081](http://localhost:8081s). Free to change localhost port if your prefer
